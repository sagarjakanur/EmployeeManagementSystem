\# Employee Management System



\## Project Objective

Build a web application to manage employees, attendance, departments, and leave requests.



\## Users

1\. Admin

2\. Employee



\## Admin Features

\- Login

\- Add Employee

\- Update Employee

\- Delete Employee

\- Manage Departments

\- View Attendance

\- Approve Leave Requests

\- Dashboard



\## Employee Features

\- Login

\- View Profile

\- Mark Attendance

\- Apply Leave

\- View Attendance History

\- View Leave Status



\## Modules

1\. Authentication

2\. Employee Management

3\. Department Management

4\. Attendance Management

5\. Leave Management

6\. Dashboard
